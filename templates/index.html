Apricot Disease Detection (Using Deep Learning)

We are a dedicated team working to develop an Apricot Disease Detector using deep learning technology in Gilgit-Baltistan. Our goal is to support local farmers by providing an easy and accurate way to identify diseases in apricot crops early, helping improve yield quality, reduce losses, and promote sustainable farming in the region.

Predict Disease

Intro:

This project focuses on detecting diseases in apricot crops using advanced machine learning techniques. The model is based on YOLOv8, a state-of-the-art object detection algorithm, which has been trained to identify and classify various apricot diseases from images. The system is integrated with a Flask-based web interface, allowing users to upload images and receive instant disease predictions. This project aims to assist farmers in early disease detection, leading to more effective crop management and improved yields.
